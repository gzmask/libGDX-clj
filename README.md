# libgdx

libgdx in clj.


## Installation

    $ mkdir classes
    $ clj -e "(compile 'gzmask.libgdx)"
    $ clj -e "(compile 'gzmask.desktop-launcher)"

## Usage


Run the project directly:

    $ clj -m gzmask.desktop-launcher

Run the project's tests (they'll fail until you edit them):

    $ clj -A:test:runner
